# ENTREGA 01: Requisitos Funcionais

**Sistema:** Sistema de Gerenciamento de Biblioteca - Controle Pessoal 
**Stack:** React + Vite (front-end) | Express + JavaScript (back-end) | PostgreSQL
 
---
 
## Back-end
 
| ID | Requisito | Descrição |
|----|-----------|-----------|
| RF-01 | Cadastro de Usuário | O sistema deve permitir cadastrar usuários com nome, e-mail e senha |
| RF-02 | Login de Usuário | O sistema deve autenticar usuários com e-mail e senha |
| RF-03 | Cadastro de Livro | O sistema deve permitir cadastrar novos livros com título, autor, gênero e quantidade |
| RF-04 | Listagem de Livros | O sistema deve listar todos os livros disponíveis no acervo |
| RF-05 | Busca de Livro | O sistema deve permitir buscar livros por título ou autor |
| RF-06 | Atualização de Livro | O sistema deve permitir editar os dados de um livro existente por ID |
| RF-07 | Exclusão de Livro | O sistema deve permitir excluir um livro do acervo por ID |
 
---
 
## Front-end
 
| ID | Requisito | Descrição |
|----|-----------|-----------|
| RF-08 | Tela de Login | O sistema deve exibir um formulário de login com campos de e-mail e senha |
| RF-09 | Tela de Cadastro | O sistema deve exibir um formulário de cadastro com campos de nome, e-mail e senha |
| RF-10 | Navegação entre telas | O sistema deve permitir navegar entre as telas de login e cadastro |
| RF-11 | Feedback de erro no login | O sistema deve exibir mensagem de erro quando o login falhar |
| RF-12 | Feedback de erro no cadastro | O sistema deve exibir mensagem de erro quando o cadastro falhar |
| RF-13 | Redirecionamento pós-login | O sistema deve redirecionar o usuário para o dashboard após login bem-sucedido |
| RF-14 | Redirecionamento pós-cadastro | O sistema deve redirecionar o usuário para a tela de login após cadastro bem-sucedido |
